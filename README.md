## 🔧 Configuration

Ashampoo PDF requires minimal setup.  settings can be configured via the **Options** menu after installation.

**Example: Setting Default Output Path**
1. Go to `File > Options > General`.
2. Under **Default Save Location**, choose your preferred folder.
3. Click **OK** to apply.

For advanced users, configuration can be adjusted via a settings file located in the user's AppData directory on Windows:
```
%APPDATA%\Ashampoo\AshampooPDF\settings.xml
```
*Editing this file directly is not recommended for most users.*
